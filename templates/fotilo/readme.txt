DESCRIPTION:

Fotilo is a clean and modern HTML template for use with WebsiteBaker CMS. 
It features a creative and clean design with an emphasis on typography, white space and attention 
to detail. Fotilo is fully responsive and retina-ready. 
It is built with clean and organized code making it very easy to customize.

There are 3 navigations:
1 Mainmenu 	_	Behind the hamburger at the right
2 Hero menu	_	Within the hero picture is room for a small menu It is displaing not only
			the menu items but also the page discriptions (don t make them to long)
			This menu is only showing at the first page! (home)
3 Footer menu	_	Usualy used for legal notice as cookies/service details etc.

Template is multilingual prepared

Hero.jpg  pic for first page (landing)
Hero2.jpg pic for all other pictures

You have all icons of fontawesome.com available <i class="fa fa_heart"><i>
